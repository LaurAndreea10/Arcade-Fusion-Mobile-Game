# Arcade Fusion Mobile Game

A mobile-focused arcade game project built as part of a personal front-end portfolio, designed for fast interaction, energetic visuals, and responsive gameplay.

## Live Demo

https://laurandreea10.github.io/Arcade-Fusion-Mobile-Game/

## Features

* Mobile-inspired arcade gameplay
* Responsive front-end layout
* Fast interaction loop
* Dynamic visual presentation
* Browser-based game experience
* GitHub Pages deployment

## Technologies

* HTML
* CSS
* JavaScript
* GitHub Pages

## Purpose

This project was created as a personal front-end experiment exploring arcade mechanics adapted for a mobile-style experience, with focus on interactivity, pacing, and visual energy.
